# Password Strength Analyzer & Cyber Education

Project ini adalah aplikasi sederhana untuk menganalisis kekuatan password menggunakan konsep **entropy**.  
Selain menampilkan tingkat kekuatan, aplikasi ini juga memberikan edukasi cyber security seperti:

- Brute force attack
- Dictionary attack
- Pentingnya hashing untuk password

## Fitur
- Hitung entropy password
- Estimasi waktu password di-crack
- Analisis komposisi karakter (huruf, angka, simbol)
- Edukasi keamanan siber

## Teknologi
- HTML / CSS / JavaScript (Frontend Only)
